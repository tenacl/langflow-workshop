# 03. 첫 번째 플로우 만들기

## 목표

이 챕터에서는 Langflow의 **Simple Agent 템플릿**을 사용하여 도구를 활용하는 AI 에이전트 챗봇을 만들어봅니다.

---

## 사전 준비

- Langflow가 실행 중인 상태 (`http://127.0.0.1:7860`)
- OpenAI API 키 (또는 다른 LLM 제공자 키)

---

## Step 1: 새 플로우 생성

1. Langflow UI에서 **New Flow** 클릭
2. 템플릿 목록에서 **Simple Agent** 선택

이 템플릿에는 다음 컴포넌트가 미리 연결되어 있습니다:

```
┌────────────┐     ┌───────────────┐     ┌─────────────┐
│ Chat Input │────▶│    Agent      │────▶│ Chat Output │
└────────────┘     │               │     └─────────────┘
                   │  ┌──────────┐ │
                   │  │Calculator│ │
                   │  └──────────┘ │
                   │  ┌──────────┐ │
                   │  │ URL Tool │ │
                   │  └──────────┘ │
                   └───────────────┘
```

- **Chat Input**: 사용자 메시지를 받는 입력 컴포넌트
- **Agent**: 질문에 따라 적절한 도구를 선택하여 실행하는 에이전트
- **Calculator**: 수학 계산을 수행하는 도구
- **URL Tool**: 웹 페이지 내용을 가져오는 도구
- **Chat Output**: 에이전트의 응답을 출력하는 컴포넌트

---

## Step 2: LLM 설정

1. **Agent** 노드에서 **Setup Provider** 클릭
2. LLM 제공자 선택 (예: OpenAI)
3. API 키가 아직 설정되지 않았다면:
   - 상단 **Settings** > **Model Providers** 이동
   - OpenAI 선택 후 API 키 입력

---

## Step 3: 모델 선택

Agent 노드의 **Language Model** 드롭다운에서 사용할 모델을 선택합니다:
- `gpt-4o` (권장)
- `gpt-4o-mini` (비용 절약)
- 또는 설정한 다른 제공자의 모델

> **참고**: 에이전트는 질문의 맥락을 파악하여 어떤 도구를 호출할지 자율적으로 결정합니다.

---

## Step 4: Playground에서 테스트

1. 화면 하단 또는 우측의 **Playground** 버튼 클릭
2. 채팅창에 메시지를 입력하여 테스트:

### 테스트 예시

**수학 계산 테스트:**
```
사용자: 4 + 4는 얼마야?
에이전트: [Calculator 도구 호출] 4 + 4 = 8입니다.
```

**웹 검색 테스트:**
```
사용자: https://news.ycombinator.com 의 첫 번째 기사 제목이 뭐야?
에이전트: [URL Tool 호출] 현재 첫 번째 기사는 "..."입니다.
```

**일반 대화:**
```
사용자: 안녕하세요!
에이전트: 안녕하세요! 무엇을 도와드릴까요?
```

---

## Step 5: 플로우 직접 만들어보기 (빈 캔버스)

템플릿 없이 직접 간단한 채팅 플로우를 만들어봅시다.

### 5-1. 빈 플로우 생성

1. **New Flow** > **Blank Flow** 선택

### 5-2. 컴포넌트 추가

좌측 사이드바에서 다음 컴포넌트를 드래그하여 캔버스에 배치합니다:

1. **Inputs** > **Chat Input**
2. **Prompts** > **Prompt**
3. **Models** > **OpenAI** (또는 다른 LLM)
4. **Outputs** > **Chat Output**

### 5-3. Prompt 템플릿 작성

Prompt 노드를 클릭하고 다음과 같이 템플릿을 작성합니다:

```
당신은 친절한 한국어 AI 어시스턴트입니다.
사용자의 질문에 명확하고 도움이 되는 답변을 제공하세요.

사용자 메시지: {user_message}
```

`{user_message}`는 **변수**입니다. 중괄호 `{}`로 감싸면 Langflow가 자동으로 입력 포트를 생성합니다.

### 5-4. 컴포넌트 연결

노드의 출력 포트를 다른 노드의 입력 포트로 드래그하여 연결합니다:

```
Chat Input ──(Message)──▶ Prompt ──(Prompt Message)──▶ OpenAI ──(Text)──▶ Chat Output
```

1. **Chat Input**의 출력 → **Prompt**의 `user_message` 입력
2. **Prompt**의 출력 → **OpenAI**의 입력
3. **OpenAI**의 출력 → **Chat Output**의 입력

### 5-5. LLM 설정 및 테스트

1. OpenAI 노드에서 API 키와 모델 설정
2. **Playground** 클릭
3. 메시지를 보내서 응답 확인!

---

## 플로우 저장 및 관리

- 플로우는 자동 저장됩니다
- 상단의 플로우 이름을 클릭하여 이름 변경 가능
- **My Projects**에서 모든 플로우를 관리할 수 있습니다

---

## 핵심 정리

| 개념 | 설명 |
|------|------|
| 템플릿 | 미리 만들어진 플로우로 빠르게 시작 |
| Agent | 도구를 자율적으로 선택하여 사용하는 컴포넌트 |
| Playground | 플로우를 실시간으로 테스트하는 환경 |
| Prompt 변수 | `{변수명}` 형식으로 동적 입력 생성 |
