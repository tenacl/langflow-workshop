## Langflow란?

[Langflow](https://www.langflow.org/)는 **오픈소스, Python 기반의 비주얼 AI 애플리케이션 프레임워크**입니다. 드래그 앤 드롭 인터페이스로 챗봇, 문서 분석기, 에이전트 등 다양한 AI 워크플로우를 코드 없이 빠르게 만들 수 있습니다.

## 사전 준비

- Python 3.10 이상
- OpenAI API 키 (또는 다른 LLM 제공자 키)
- 웹 브라우저 (Chrome, Firefox 등)

## 학습 순서

1. **01-intro** → 개념 이해
2. **02-setup** → 설치 및 실행
3. **03-basic-flows** → 첫 플로우 만들기
4. **04-components** → 컴포넌트 깊이 이해
5. **05-advanced** → API 연동 및 커스텀 컴포넌트
6. **06-exercises** → 실습으로 복습
